# n8n

## Docker Commands: (cd into folder contain file `docker-compose.yml`, `docker-compose.override.yml`)

-   Up & running:

```Powershell
docker-compose -f docker-compose.yml -f docker-compose.override.yml up -d
```

-   Stop & Removing:

```Powershell
docker-compose down
```

-   Update:

```Powershell
docker-compose pull
```
